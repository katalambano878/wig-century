# E-commerce Store

A Next.js e-commerce application with Supabase backend, product catalog, cart, checkout, and admin dashboard.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables (copy from `.env.example` if present, or create `.env.local` with your Supabase and app URL values).

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Production build
- `npm run lint` — Run ESLint
- `npm run db:migrate` — Run database migrations
- `npm run create-admin` — Create an admin user (see script for usage)

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Database / Auth:** Supabase
- **Styling:** Tailwind CSS
- **Language:** TypeScript

Customize branding, content, and environment variables for your own store.
